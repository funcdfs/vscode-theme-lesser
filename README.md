## Install

search **"lesser"** in VS Code Extensions (`Cmd+Shift+X` / `Ctrl+Shift+X`)

## Semantic Highlighting

Lesser 提供完整的语义化高亮支持，采用自然色系配色，每种语义元素都有独特的颜色：

**支持的语言**: JavaScript, TypeScript, React (JSX/TSX), Python, Rust, Go, C++, Dart, Vue
